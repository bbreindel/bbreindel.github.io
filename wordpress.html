Wordpress Business Site

https://www.farmhouseonthepond.com/

This is a Wordpress site that I built to support my wife’s business. Last year, she began selling baked goods at a farmers market as well as through direct customer orders.

The checkout page contains a significant amount of custom Javascript and Jquery, including functionality that checks zip code in order to determine if delivery is available for a specific customer location. Additionally, I completely rebuilt a Wordpress add-on to allow the use of two different calendars with separate lookup functionality and settings for order pickup and order delivery.
